## Ordered by Length
### Local Rules
`local15`: Smallest rule. Unrelated to `50-30k`.

`local50-30k`: Iterative rules designed for consistent coverage. Hand-selected to ensure efficient tasks. Each rule set is built off the last.

`local50k-250k`: Uses `local30k` as a foundation then iteratively builds for coverage.Preference based selection by adding top selection of other rules.

- Beyond `250k` contains non-repository rules created by myself

### Other Rules
`post`: Focused rules for "post-processing" after base changes.

`phrase`: Focused rules for phrases using 2-3 word phrases. Uses multiple `ptt` modes

## Ordered by Performance

`append`: Append rules from frequency sorted data. Non-alpha append rules.

`prepend`: Prepend rules from frequency sorted data. Alpha prepend rules.

`taggle`: Append and toggle rules in one from frequency sorted data.

`poggle`: Prepend and toggle rules in one from frequency sorted data.
